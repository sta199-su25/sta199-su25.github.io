Computing summary stats

  • With summarize: condenses information into smaller result table

  • With mutate: adds a new column to the data

  • With groupby: results for each provided group

Sample Data

df <- tibble(
    C1 = c("A", "A", "A", "B", "B"),
    C2 = c("X", "Y", "X", "X", "Y"),
    C3 = 1:5
)

df
# A tibble: 5 × 3
  C1    C2       C3
  <chr> <chr> <int>
1 A     X         1
2 A     Y         2
3 A     X         3
4 B     X         4
5 B     Y         5
