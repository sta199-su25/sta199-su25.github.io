What about the pipe?

The following two pieces of code are equivalent:

left_join(df_X, df_Y, 
          by = join_by(id_X == id_Y))



df_X |> left_join(df_Y, 
                  by = join_by(id_X == id_Y))

Letâ€™s save!

Most often, you will want to save the result of a join to a new data frame.

df_X_Y <- df_X |> left_join(df_Y, 
                            by = join_by(id_X == id_Y))
